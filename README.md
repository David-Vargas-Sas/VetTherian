# 🐾 VetTherian - Sistema Veterinario

## 📌 Descripción

VetTherian es una aplicación web básica desarrollada en HTML, CSS y JavaScript que permite gestionar información de una veterinaria.

Incluye módulos para:

* 🐶 Mascotas
* 👤 Propietarios
* 🩺 Veterinarios

La aplicación utiliza **LocalStorage** como almacenamiento temporal y está enfocada en prácticas de frontend.

## 🚀 Funcionalidades

### 🔐 Login

* Validación de credenciales
* Persistencia de sesión con LocalStorage
* Redirección automática

### 🐶 Mascotas

* Crear, editar y eliminar mascotas
* Asignación de propietario
* Visualización en tabla dinámica

### 👤 Propietarios

* Registro de propietarios
* Validaciones básicas
* CRUD completo

### 🩺 Veterinarios

* Registro de veterinarios
* Gestión de especialidades y contacto
* CRUD completo

---
